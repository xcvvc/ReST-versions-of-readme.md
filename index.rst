xcvvc
=====

code on: https://github.com/xcvvc

xcvvc is a collection of scripts.
yf: a data grab from the financial web portal (yahoo).
pics: a set of scripts to archive photo collections from iPhoto.
tunes: a set of scripts to archive mp3 collections from iTunes.
epublish: reference copies of navigation and table of contents files for Amazon Kindle.


Instructions: As these scripts are put into use only quarterly or annually, open the script, update the desired file and folder names directly into the script and then run.  There is no argv(), no .obj executable., no installation.
